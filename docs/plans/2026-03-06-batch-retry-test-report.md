## 测试环境

- Backend: http://localhost:8000
- Browser: Chrome/Firefox/Edge (推荐)

## 测试前准备
1. 确保后端服务已启动：`cd yt-dlp-api && python app/main.py`
2. 确保前端服务已启动：`cd yt-dlp-api-front && npm start`
3. 准备一些失败的任务用于测试（可以通过提交无效 URL 创建）
